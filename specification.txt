Project File Arrangement:

Deprecated: if a module is never used, put it here

gui: all module, related to QT

util: if a module aims to solve a problem (called utilities ), it should be placed here. Such as icmp api, or checksum modules.

root directory: only the main drive of the program, or docs should be placed here.
